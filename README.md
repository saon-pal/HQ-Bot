# HQ-Bot
Bot that can guess answers for the the live trivia game, HQ

## Prerequisites
HQ-Bot needs the following python3 modules:
```
bs4
PIL
lxml
pytesseract
re
requests
time
```

## Usage
Add the following line to the end of bot.py:
```python
guessAnswer('examplePath/example.png')
```
